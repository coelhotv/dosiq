# AP-97 — CSS token definitions duplicadas/conflitantes em múltiplos arquivos

**Category:** Css
**Status:** active
**Related Rule:** R-118
**Applies To:** css, design-system, tokens

## Problem

Múltiplos arquivos definem o mesmo token com valores diferentes. A última definição carregada vence e sobrescreve o design system canônico em runtime.

## Prevention

Centralizar tokens em uma fonte canônica por domínio visual. Antes de mover ou migrar tokens, localizar todas as definições existentes e eliminar duplicatas conflitantes no mesmo patch.
