## Regra

Ao configurar `Notifications.setNotificationHandler()`, usar `shouldShowBanner` e `shouldShowList` em vez do depreciado `shouldShowAlert` (depreciado no SDK 53+).

## Como aplicar

```js
Notifications.setNotificationHandler({
  handleNotification: async () => ({
    shouldShowBanner: true,   // banner flutuante em foreground
    shouldShowList: true,     // aparece no notification center do OS
    shouldPlaySound: true,
    shouldSetBadge: false,
  }),
})
```

## Por que

`shouldShowAlert: true` gera warning no console do Expo e será removido em versão futura. `shouldShowBanner` + `shouldShowList` são os substitutos diretos com comportamento equivalente.
