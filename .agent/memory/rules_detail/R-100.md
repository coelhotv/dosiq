# R-100: Hook Dependencies Array Precision

**Summary:** Ensure useEffect/useMemo dependencies are precise and complete

**Applies to:** react

**Tags:** react, performance

## Rationale

Incorrect dependency arrays cause stale closures, infinite loops, or missed updates. ESLint's react-hooks/exhaustive-deps catches most issues.
