# R-122: Database Query Result Validation

**Summary:** Validate query results match expected schema before using

**Applies to:** ts, supabase

**Tags:** testing, zod

## Rationale

Database schemas can change or return unexpected results. Validate with Zod before consuming query results in application logic.
