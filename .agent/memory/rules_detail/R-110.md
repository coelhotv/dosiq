# R-110: Guard Clauses AFTER All Hooks

**Summary:** Early returns with guard clauses MUST come AFTER all hook declarations (States → Memos → Effects → Handlers). Placing guard clause before hooks causes 'React Hook called conditionally' error.

**Applies to:** react

**Tags:** react, safety

## Rationale

Conditional early returns before hook declaration break React's hook ordering guarantees and can crash the component.

## Implementation Notes

Declare all hooks first, then apply guard clauses before render output.
