# R-095: Wave 1 Component Purity

**Summary:** Ensure components render consistently without side effects

**Applies to:** react

**Tags:** react, testing

## Rationale

Pure components are predictable and testable. Avoid direct DOM manipulation, random values, or external API calls in render logic.
