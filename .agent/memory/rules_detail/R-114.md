# R-114: Dose Rescheduling Constraints

**Summary:** Prevent rescheduling doses beyond 7-day lookback window

**Applies to:** ts

**Tags:** supabase, process

## Rationale

Doses older than 7 days should not be rescheduled to prevent historical data manipulation. Enforce this at database level with check constraints.
