# R-092: Spec Path Verification

**Summary:** Verify spec file paths exist before importing specifications

**Applies to:** react, ts

**Tags:** testing, react

## Rationale

During development, spec files may be referenced but not yet created. Always verify spec paths exist before importing to catch file-not-found errors early in tests.
