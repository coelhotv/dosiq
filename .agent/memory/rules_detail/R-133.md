# R-133: Inspect Component Prop Interface Before Copying

**Summary:** Antes de reutilizar/copiar um componente entre contextos, SEMPRE ler a assinatura de destructuring para confirmar os props esperados. Não assuma nomes por semântica.

**Applies to:** react

**Tags:** react, safety

## Rationale

Prop names that sound equivalent often are not. Copying usage by intuition creates silent breakage.

## Implementation Notes

Inspect the component signature directly before reusing it in a new context.
