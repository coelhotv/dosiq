# R-094: CSS Selector Stability

**Summary:** Use stable class names and data attributes for CSS selectors

**Applies to:** react

**Tags:** design, react

## Rationale

CSS selectors should be resilient to HTML structure changes. Prefer data attributes over fragile nth-child or deep descendant selectors.
