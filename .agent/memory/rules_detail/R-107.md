# R-107: Notification Scheduling Precision

**Summary:** Use server-side scheduling for timezone-aware notifications

**Applies to:** ts, supabase

**Tags:** telegram, supabase

## Rationale

Client-side notifications fail when app is closed. Server-side scheduling (cron jobs, queues) ensures reliable timezone-aware delivery.
