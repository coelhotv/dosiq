# R-146: Centralize PDF Layout Geometry

**Summary:** Em geradores de PDF, coordenadas, offsets, larguras e alturas DEVEM ser extraídas para constantes nomeadas. Sem números mágicos inline.

**Applies to:** all

**Tags:** design, process

## Rationale

Inline geometry values make PDF layout changes brittle and hard to audit.

## Implementation Notes

Move PDF geometry into named constants before expanding or tuning the layout.
