# R-146: Button Size Consistency

**Summary:** Use standardized button sizes (small, medium, large) across designs

**Applies to:** react

**Tags:** design

## Rationale

Inconsistent button sizes confuse users about interaction targets. Define standard sizes and apply consistently.
