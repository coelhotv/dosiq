# R-156: AdherenceLabel usa linguagem humana para modo simple

**Summary:** Use human language (not %) in simple mode: 'em dia', 'perdidas', 'em risco'

**Applies to:** react

**Tags:** design

## Rationale

Dona Maria reads verbs, not percentages. Thresholds: >90% 'em dia', 70-90% 'algumas perdidas', 50-70% 'em risco', <50% 'muitas perdidas', 0% hidden.
