# R-104: || vs ?? para Valores Numéricos com Zero Válido

**Summary:** Use ?? (nullish coalescing) when 0 is a valid value. || treats 0 as falsy and replaces with fallback — incorrect behavior.

**Applies to:** all

**Tags:** data, safety

## Rationale

Numeric zero is valid business data in many flows. Using `||` silently mutates that value and produces wrong calculations or UI.

## Implementation Notes

Use `??` whenever the fallback should apply only to `null` or `undefined`, not to `0`.
