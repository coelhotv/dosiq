# R-131: Timezone Offset Consistency

**Summary:** Always use UTC internally; convert to user timezone only at UI boundaries

**Applies to:** ts, react

**Tags:** timezone, react

## Rationale

Mixing timezone contexts causes subtle bugs. Enforce UTC internally and convert only for display/input.
