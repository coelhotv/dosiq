# R-125: Batch Operation Atomicity

**Summary:** Ensure batch operations complete as atomic units

**Applies to:** ts, supabase

**Tags:** supabase, process

## Rationale

Partial batch completion causes inconsistent state. Use transactions or compensating operations for atomicity.
