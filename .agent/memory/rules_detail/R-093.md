# R-093: ESLint Configuration Consistency

**Summary:** Maintain consistent ESLint rules across all components

**Applies to:** js, react

**Tags:** process, react

## Rationale

ESLint enforces code quality and consistency. Disabling rules should be rare and documented. Prefer fixing the code to disabling the rule.
