# R-105: Medicine Search Indexing Strategy

**Summary:** Index medicine names and ingredients for fast autocomplete search

**Applies to:** supabase

**Tags:** supabase, performance

## Rationale

Full-text search performance matters for UX. Database indexes on medicine names and ingredients enable fast autocomplete without client-side filtering.
