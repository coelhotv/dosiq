# R-113: Protocol Pause Impact Assessment

**Summary:** Calculate adherence impact before allowing protocol pause

**Applies to:** ts, react

**Tags:** process, react

## Rationale

Pausing a protocol mid-cycle affects adherence metrics. Users should see impact before confirming pause to prevent accidental disruption.
