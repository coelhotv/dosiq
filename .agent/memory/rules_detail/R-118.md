# R-118: Multi-Dose Days Handling

**Summary:** Handle protocols with multiple doses per day explicitly

**Applies to:** ts, react

**Tags:** supabase, react

## Rationale

Some medicines require multiple doses daily. Ensure scheduling, reminders, and adherence calculations handle multi-dose days correctly.
