# R-138: Icon-Text Pairing

**Summary:** lucide-react icons NUNCA aparecem sozinhos. SEMPRE acompanhados de texto label. Exception: icon-only MUST have aria-label + title.

**Applies to:** react

**Tags:** design, react

## Rationale

Icons alone are ambiguous for many users. Pairing with text reduces guesswork and improves accessibility.

## Implementation Notes

If an icon must stand alone, provide explicit accessibility labeling.
