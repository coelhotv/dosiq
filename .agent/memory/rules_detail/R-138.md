# R-138: Color Contrast Accessibility

**Summary:** Maintain WCAG AA contrast ratios (4.5:1 minimum) for all text

**Applies to:** react

**Tags:** design

## Rationale

Low contrast colors are unreadable for users with vision deficiencies. Verify contrast ratios in design and implementation.
