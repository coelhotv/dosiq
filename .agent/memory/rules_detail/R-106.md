# R-106: Protocol Schedule Normalization

**Summary:** Normalize protocol schedules to canonical time zones during storage

**Applies to:** ts, supabase

**Tags:** timezone, supabase

## Rationale

User time zones vary. Storing all schedules in UTC and converting on display prevents ambiguity and timezone-related bugs.
