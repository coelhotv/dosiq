# R-098: DoseZoneList Adapter Pattern — Interface Mismatch

**Summary:** When a new component has a different callback interface than existing handlers, create thin adapter functions. Do NOT refactor existing handlers.

**Applies to:** react

**Tags:** react

## Rationale

Interface mismatches between old and new components are common during refactors. Thin adapters localize the compatibility layer and avoid unnecessary churn in stable handlers.

## Implementation Notes

Wrap the existing handler with a translation layer near the consuming component instead of rewriting shared logic.
