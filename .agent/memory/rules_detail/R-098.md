# R-098: Component Interface Contracts

**Summary:** Define and document component prop interfaces explicitly

**Applies to:** react, ts

**Tags:** react, testing

## Rationale

Component contracts (props) should be explicit and stable. Changes to component interfaces are breaking changes that affect consumers.
