# R-132: Serverless Function Timeout Management

**Summary:** Monitor serverless function execution time and abort long-running operations

**Applies to:** ts, vercel

**Tags:** infra, vercel

## Rationale

Vercel functions have timeout limits. Long operations should be moved to background jobs or paginated.
