# R-112: Stock Movement Audit Trail

**Summary:** Log all stock changes (purchases, usage, adjustments) with reason and timestamp

**Applies to:** supabase

**Tags:** supabase, process

## Rationale

Stock reconciliation requires audit trails. Every change must be logged with reason and actor for accountability and debugging.
