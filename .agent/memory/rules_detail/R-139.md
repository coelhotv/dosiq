# R-139: Spacing and Alignment Grid

**Summary:** Use consistent spacing unit (e.g., 8px) for alignment across design

**Applies to:** react

**Tags:** design

## Rationale

Consistent spacing creates visual harmony. Enforce grid-based spacing to ensure alignment and professional appearance.
