# R-139: Font @import Scoping Strategy

**Summary:** Google Fonts SEMPRE via CSS @import em token files (scoped), NUNCA via <link> em index.html (global). Fonts ficam scoped ao [data-redesign='true'].

**Applies to:** all

**Tags:** design, performance

## Rationale

Scoped font loading limits global side effects and keeps redesign typography isolated.

## Implementation Notes

Load fonts from CSS token layers instead of global HTML tags when the typography system is intentionally scoped.
