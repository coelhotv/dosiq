# R-101: Adherence Calculation Logic Isolation

**Summary:** Keep adherence calculation in dedicated service module

**Applies to:** ts

**Tags:** supabase, process

## Rationale

Adherence logic is complex and reused across dashboard, reports, and notifications. Isolation in dedicated service prevents duplication and bugs.
