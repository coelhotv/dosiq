# R-101: DoseZoneList Internal Sub-components

**Summary:** DoseCard and ZoneSection are internal sub-components of DoseZoneList, NOT exported. No use outside DoseZoneList.

**Applies to:** react

**Tags:** react

## Rationale

These sub-components exist to support one parent component. Exporting them would create accidental public API surface and tighter coupling.

## Implementation Notes

Keep `DoseCard` and `ZoneSection` private to `DoseZoneList` unless there is an explicit API decision to promote them.
