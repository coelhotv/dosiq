# R-096: Framer Motion SVG — Set strokeDashoffset in Both style and initial

**Summary:** For SVG motion.circle with strokeDashoffset animation, set value in BOTH style (static CSS) and initial/animate. Without style, browser renders full ring before Framer Motion takes control.

**Applies to:** react

**Tags:** react, design, performance

## Rationale

SVG strokes can flash in the wrong state before animation mounts. Defining the same baseline in static style and animation props avoids the visual glitch.

## Implementation Notes

When animating SVG ring progress, set `strokeDashoffset` in `style` and in the Framer Motion lifecycle props.
