# R-096: Framer Motion SVG Animation

**Summary:** Use animatePresence and motion.div wrappers for SVG animations

**Applies to:** react

**Tags:** design, react, performance

## Rationale

Framer Motion requires AnimatePresence context for exit animations. SVG animations must be wrapped in motion.div for transform properties to work.
