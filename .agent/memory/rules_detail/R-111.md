# R-111: Use calculateExpectedDoses() for Protocol Consumption

**Summary:** Always use calculateExpectedDoses(protocols, days) from @utils/adherenceLogic. calculateDailyIntake() ignores frequency and causes inflated estimates for non-daily protocols.

**Applies to:** all

**Tags:** data, supabase

## Rationale

Expected-dose math must respect protocol frequency. Using the wrong helper inflates consumption estimates and distorts adherence calculations.

## Implementation Notes

For protocol consumption and expected-dose counts, use `calculateExpectedDoses()` as the canonical helper.
