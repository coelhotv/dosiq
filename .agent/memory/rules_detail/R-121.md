# R-121: Service Parameter Validation

**Summary:** Validate all service parameters at entry points before use

**Applies to:** ts, supabase

**Tags:** testing, supabase

## Rationale

Invalid parameters cause silent failures or crashes. Validate early at service entry points and return clear error messages.
