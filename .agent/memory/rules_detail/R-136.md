# R-136: Callback Batch Operations Must Match UI Promise

**Summary:** When UI says 'Confirmar 3 doses', implementation must process all 3, not just 1

**Applies to:** ts

**Tags:** testing, process

## Rationale

Wave 6.5: onRegisterAll(doses) only registered doses[0], violating UI contract. Test batch callbacks explicitly.

This rule is part of the DEVFLOW migration (Wave 1C). Refer to the main rules index for additional context and related patterns.
