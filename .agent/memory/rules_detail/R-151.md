# R-151: Modal Component Consistency — Use Shared Modal

**Summary:** Ao implementar modals, sempre usar o <Modal> compartilhado do projeto. Nunca criar overlay divs customizados. Shared modal tem Esc key handling, backdrop e animações consistentes.

**Applies to:** react

**Tags:** react, design

## Rationale

Custom overlays drift in behavior and styling. The shared modal preserves accessibility and consistency.

## Implementation Notes

Prefer the shared modal primitive for all new modal flows unless there is an explicit exception.
