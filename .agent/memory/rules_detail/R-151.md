# R-151: Modal Component Consistency — Use Shared Modal, Not Custom Overlays

**Summary:** Always use project's shared Modal component for dialogs

**Applies to:** react

**Tags:** design, react

## Rationale

Custom modals diverge in behavior and styling. Shared Modal ensures consistent UX (Esc key, animations, backdrop).
