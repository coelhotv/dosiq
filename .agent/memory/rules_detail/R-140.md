# R-140: Heading Hierarchy Variables

**Summary:** TODOS os níveis h1-h6 DEVEM ter variáveis específicas de size + weight em CSS tokens. Sem variáveis → hard-coded values → inconsistência em redesigns futuros.

**Applies to:** all

**Tags:** design

## Rationale

Heading hierarchy is foundational design-system infrastructure. Hardcoded values make later redesign work inconsistent and expensive.

## Implementation Notes

Define explicit token variables for each heading level and weight pair.
