# R-129: Race Condition Prevention

**Summary:** Use request debouncing/throttling for high-frequency operations

**Applies to:** ts, react

**Tags:** performance, react

## Rationale

Rapid operations (filtering, searching) can cause race conditions. Debounce/throttle to ensure operations complete in order.
