# R-108: Telegram Bot State Persistence

**Summary:** Persist Telegram bot conversation state in database between messages

**Applies to:** telegram, supabase

**Tags:** telegram, supabase

## Rationale

Telegram bots are stateless. Storing conversation context in database enables multi-step workflows and prevents state loss on server restart.
