# R-003 — Import Existence Check [HIGH]

## Rule
Never import files that do not exist. Always validate with `npm run build` before pushing.

## Why
ERR_MODULE_NOT_FOUND from orphaned `retryManager` import that passed linting but failed at runtime. Build validation catches missing imports that linters miss.

## Enforcement
```bash
npm run build  # Must pass before any push
```

## Consolidated anti-patterns
- `AP-003` — import a file that doesn't exist

Use this rule as the canonical guidance for missing-import prevention. The AP remains only as a compact negative lookup.
