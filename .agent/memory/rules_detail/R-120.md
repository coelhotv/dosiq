# R-120: CSS Selector Consolidation Pattern

**Summary:** When 2+ classes share >80% of styles, consolidate via selector grouping (,). Separate only size/state modifiers.

**Applies to:** all

**Tags:** design

## Rationale

Duplicated CSS drift over time. Consolidating near-identical selectors reduces maintenance and keeps variants intentional.

## Implementation Notes

Group shared selectors and isolate only the modifiers that truly differ.
