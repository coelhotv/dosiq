# R-120: User Preference Persistence

**Summary:** Persist user preferences (theme, language, notification settings) per user

**Applies to:** supabase, react

**Tags:** supabase, design

## Rationale

Preferences are user-specific and should survive across sessions. Store in database, not localStorage, to sync across devices.
