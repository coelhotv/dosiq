# R-109: API Rate Limiting Aware Design

**Summary:** Design client code to batch requests and respect rate limits

**Applies to:** ts, react

**Tags:** performance, process

## Rationale

Third-party APIs have rate limits. Client code should batch requests and implement exponential backoff for resilience.
