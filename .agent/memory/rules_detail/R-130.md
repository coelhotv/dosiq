# R-130: String Comparison for Dates

**Summary:** Compare ISO date strings directly for simple date ordering

**Applies to:** ts

**Tags:** process, testing

## Rationale

ISO 8601 format is sortable as strings. For simple comparison, avoid Date object overhead. Use proper Date objects for arithmetic.
