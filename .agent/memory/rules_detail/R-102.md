# R-102: Protocol Consumption State Management

**Summary:** Use state machines for protocol consumption workflows

**Applies to:** react, ts

**Tags:** process, react

## Rationale

Protocol consumption has multiple states (pending, active, completed, paused). State machines prevent invalid state transitions and clarify logic.
