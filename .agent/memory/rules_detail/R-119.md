# R-119: Conditional Dose Patterns

**Summary:** Support protocols with conditional dose patterns (e.g., as-needed)

**Applies to:** ts

**Tags:** supabase, process

## Rationale

Not all medicines follow fixed schedules. Support flexible patterns (as-needed, PRN) with appropriate tracking and adherence rules.
