# R-127: Select Mínimo em Queries Supabase para Listas

**Summary:** Queries para listas DEVEM selecionar apenas colunas usadas no render. Resultado: ~76-80% redução de payload em timelines.

**Applies to:** supabase

**Tags:** performance, supabase

## Rationale

Selecting unused columns wastes bandwidth and slows down list-heavy views.

## Implementation Notes

Keep `select(...)` clauses minimal and tied to actual render needs.
