# R-127: Logging Sensitive Data Masking

**Summary:** Never log sensitive data (passwords, tokens, PII); mask or omit

**Applies to:** ts

**Tags:** infra, process

## Rationale

Logs may be archived or shared. Prevent accidental exposure of sensitive data by implementing strict masking rules.
