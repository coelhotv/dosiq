# R-097: CSS color-mix Progressive Enhancement

**Summary:** Use color-mix with fallback syntax for browser compatibility

**Applies to:** react

**Tags:** design, performance

## Rationale

color-mix() is modern but not universally supported. Always provide fallback colors or use @supports queries for progressive enhancement.
