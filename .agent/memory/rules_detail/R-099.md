# R-099: Adapter Pattern for Service Integration

**Summary:** Use adapter patterns to decouple components from specific services

**Applies to:** react, ts

**Tags:** process, testing

## Rationale

Adapters allow components to work with different service implementations. This enables testing with mock services without changing component code.
